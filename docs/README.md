## API Docs

Interactive docs available at `http://localhost:8000/docs` after starting the server.

To import into your API client:
- **Postman**: Import → Link → `http://localhost:8000/openapi.yaml`
- **Bruno**: Import → OpenAPI → `http://localhost:8000/openapi.yaml`
- **Insomnia**: Import → URL → `http://localhost:8000/openapi.yaml`

Or use the local file: `docs/openapi.yaml`
